# fig-step

A JavaScript library to add numbered annotations to elements on a web page.

## Demo
- https://code4fukui.github.io/fig-step/?fig=on

## Features
- Easily add numbered annotations to HTML elements
- Annotations can be shown or hidden with a URL parameter
- Customizable appearance of annotations

## Usage
To use `fig-step`, include the JavaScript file in your HTML:

```html
<script type="module" src="https://code4fukui.github.io/fig-step/fig-step.js"></script>
```

Then, add the `<fig-step>` element wherever you want an annotation to appear:

```html
<fig-step>1 Do this thing</fig-step>
```

The number in the annotation will be automatically generated.
